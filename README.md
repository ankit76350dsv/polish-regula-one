# RegulaOne

## Unified Compliance Platform for Polish Businesses

RegulaOne is an enterprise compliance platform designed specifically for the Polish market.  
The platform helps companies manage legal, HR, financial, environmental, and data protection compliance requirements in one centralized system.

The goal of RegulaOne is to simplify complex regulatory processes and automate compliance workflows required by Polish and European laws.

---

# Why RegulaOne Exists

Businesses in Poland are required to comply with multiple government regulations and reporting systems, including:

- KSeF e-Invoicing
- GDPR / RODO
- BDO Waste Reporting
- Labour Code attendance tracking
- BHP workplace safety compliance
- Whistleblower protection regulations

Most companies currently use:
- spreadsheets
- manual paperwork
- disconnected software tools
- email-based workflows

This creates:
- compliance risks
- missed deadlines
- audit issues
- security problems
- high administrative costs

RegulaOne solves this problem by providing a single platform for compliance management.

# Core Applications

---

# 1. KSeFPilot — E-Invoice System

### Purpose
Allows companies to create and send invoices directly to the Polish KSeF government system.

### Key Features
- Create FA(3) XML invoices
- Send invoices to KSeF API
- Digital certificate authentication
- UPO receipt storage
- Offline invoice mode
- QR verification support

### Compliance
- Polish KSeF Mandate
- 10-year invoice retention

---

# 2. TimePilot — Time Tracking System

### Purpose
Tracks employee working hours and attendance.

### Key Features
- Clock in / clock out
- GPS attendance tracking
- Break logging
- Overtime calculation
- Monthly payroll reports
- Push notifications

### Compliance
- Polish Labour Code
- Attendance record requirements

---

# 3. CompliancePilot — HR Compliance System

### Purpose
Tracks employee compliance documents and workplace safety records.

### Key Features
- Medical certificate tracking
- BHP training tracking
- Expiry alerts
- Compliance dashboard
- Secure document uploads
- Work blocking for expired certificates

### Compliance
- BHP workplace safety regulations
- Labour safety law

---

# 4. SafeVoice — Whistleblower System

### Purpose
Allows employees to anonymously report illegal or unethical activities.

### Key Features
- Anonymous reporting
- AES-256 encrypted reports
- PIN-based tracking
- Admin review dashboard
- Status updates
- Secure attachments

### Compliance
- Polish Whistleblower Protection Law

---

# 5. EcoPilot — BDO Waste Reporting

### Purpose
Helps companies manage environmental waste reporting obligations.

### Key Features
- Monthly waste entry
- Annual report generation
- XML export
- PDF export
- Historical logs
- BDO number management

### Compliance
- BDO Waste Database regulations
- Environmental reporting requirements

---

# 6. PrivacyPilot — GDPR / RODO Compliance

### Purpose
Helps businesses comply with GDPR and Polish RODO regulations.

### Key Features
- Processing activity wizard
- GDPR Register generation
- Privacy policy generation
- DPIA detection
- PDF/DOCX exports
- Audit logs

### Compliance
- GDPR / RODO
- EU data protection requirements
